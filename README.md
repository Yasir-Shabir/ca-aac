# CA-AAC: Context-Aware Adaptive Access Control System

[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Overview

Context-Aware Adaptive Access Control System combining:
- **Rabin Logic** for formal verification
- **Machine Learning** for threat estimation  
- **TON_IoT Dataset** for empirical validation

## Quick Start
```bash
# Clone repository
git clone https://github.com/Yasir-Shabir/ca-aac.git
cd ca-aac

# Install dependencies
pip install -r requirements.txt

# Run analysis
python src/ca_aac_analysis.py --dataset data/Train_Test_Network.csv
```

## Docker
```bash
# Build and run
docker-compose up
```
